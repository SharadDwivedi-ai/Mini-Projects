# 📦 JavaScript Mini Projects Collection

This repository contains a collection of beginner-friendly JavaScript projects designed to strengthen core web development concepts like DOM manipulation, event handling, and problem-solving.

---

## 🚀 Projects

### 🎨 Color Changer
A simple interactive app that changes the background color when a button is clicked.

**Features:**
- Random color generation
- Instant UI update
- Button-based interaction

**Concepts Used:**
- DOM Manipulation
- Event Handling
- Functions

---

### ⚖️ BMI Calculator
A web application that calculates Body Mass Index (BMI) based on user input.

**Features:**
- Input fields for height and weight
- BMI calculation using formula
- Displays health category

**Concepts Used:**
- Form Handling
- Input Validation
- Conditional Logic
- Type Conversion

---

### ⏰ Digital Clock
A real-time digital clock that displays the current time.

**Features:**
- Live time updates every second
- Simple and clean UI

**Concepts Used:**
- JavaScript Date Object
- setInterval()
- DOM Updates

---

### 🎯 Guess the Number Game
An interactive game where the user guesses a randomly generated number.

**Features:**
- Random number generation
- User input validation
- Feedback system (Too High / Too Low)
- Attempt tracking

**Concepts Used:**
- Math.random()
- Conditional Statements
- Game Logic
- State Management

### Todo App - Task Management
This application allows users to add, edit, and delete tasks, helping them stay organized and productive.

**Features:**
- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- 🔄 Real-time UI updates
- 💾 Persistent storage using LocalStorage

**Concepts Used:**
- DOM Manipulation
- Event Handling
- Event Delegation
- State Management
- Dynamic Rendering
- Array Methods (push, splice, map, filter)

###  Weather App – API Based Project
The Weather App allows users to search for any city and view its current weather information.
**📌 Features**
- 🔍 Search weather by city name
- 🌡️ Display temperature and weather conditions
- 🌥️ Show weather description (e.g., clear, cloudy, rainy)
- ⚡ Real-time data fetching using API
- ❌ Error handling for invalid city names
- 🔄 Dynamic UI updates

**🎯 Concepts Covered**
- API Integration (Fetch API)
- Async/Await & Promises
- JSON Data Handling
- DOM Manipulation
- Event Handling
- Error Handling

### 📝 Notes App – CRUD Implementation
This project focuses on managing data (CRUD Implementation), updating the UI dynamically, and persisting data using LocalStorage. 
** 📌 Features **
- ➕ Add new notes
- 📋 Display all notes
- ✏️ Edit existing notes
- ❌ Delete notes
- 🔄 Real-time UI updates
- 💾 Persistent storage using LocalStorage

**🎯 Concepts Covered**
- CRUD Operations
- DOM Manipulation
- Event Delegation
- State Management
- Dynamic Rendering
- LocalStorage
- Array Methods (push, splice, forEach)
