# 📦 JavaScript Mini Projects Collection

This repository contains a collection of beginner-friendly JavaScript projects designed to strengthen core web development concepts like DOM manipulation, event handling, and problem-solving.

---

## 🚀 Projects

### 🎨 Color Changer
A simple interactive app that changes the background color when a button is clicked.

**Features:**
- Random color generation
- Instant UI update
- Button-based interaction

**Concepts Used:**
- DOM Manipulation
- Event Handling
- Functions

---

### ⚖️ BMI Calculator
A web application that calculates Body Mass Index (BMI) based on user input.

**Features:**
- Input fields for height and weight
- BMI calculation using formula
- Displays health category

**Concepts Used:**
- Form Handling
- Input Validation
- Conditional Logic
- Type Conversion

---

### ⏰ Digital Clock
A real-time digital clock that displays the current time.

**Features:**
- Live time updates every second
- Simple and clean UI

**Concepts Used:**
- JavaScript Date Object
- setInterval()
- DOM Updates

---

### 🎯 Guess the Number Game
An interactive game where the user guesses a randomly generated number.

**Features:**
- Random number generation
- User input validation
- Feedback system (Too High / Too Low)
- Attempt tracking

**Concepts Used:**
- Math.random()
- Conditional Statements
- Game Logic
- State Management

---

## 🛠️ Technologies Used
- HTML5

- JavaScript (ES6)
